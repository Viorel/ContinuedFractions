# Continued Fractions

An experimental tool to convert decimal numbers or fractions to continued fractions, and vice versa.

For example:

Entered number   | The result
:---             | :--- 
12.34		     | [ 12; 2, 1, 16 ]
5/6              | [ 0; 1, 5 ]
0.8(3)           | [ 0; 1, 5 ]

Entered continued fraction | The result 
:---                | :--- 
[ 12; 2, 1, 16 ]    | 12.34
[ 0; 1, 5 ]         | 0.8(3)

The program accepts various kinds of numbers:

Input               | Meaning
:---                | :--- 
123                 | Integer number
12.34               | Decimal number
0.8(3)              | Repeating (recurring) decimal
5/6                 | Fraction
1.2e-7              | Scientific notation (1.2×10&#x207B;&#x2077;)
pi | Number &#x03C0; (3.1415926&#x2026;)
e | Euler's (Napier's) number (2.7182818&#x2026;)
